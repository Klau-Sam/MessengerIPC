Opis poszczegolnych plikow:
    server.c: Program serwer, ktory swoja dzialalnosc rozpoczyna od wczytywania danych o klientach oraz o grupach z pliku konfiguracyjnego. Nastepnie, w nieskonczonej petli, serwer wychwytuje zadania wysylane od klienta poprzez kolejki komunikatow. Serwer przyjmuje zadania, weryfikuje je i realizuje operacje obejmujace m.in. logowanie na serwer, wylogowywanie, mozliwosc zapisania/wypisania sie z jednej do/z grup tematycznych oraz przesylanie wiadomosci do wszystkich uzytkownikow nalezacych do danej grupy, a takze bezposrednia komunikacje miedzy uzytkownikami w bazie danych.
    client.c: Program klient, ktory przy pomocy kolejek komunikatow realizuje swoje potrzeby zawierajace sie wsrod przyjaznego dla uzytkownika menu. Wybor wybranego zadania dla serwera dokonuje sie poprzez wpisanie odpowiedniej liczby. Oczywiscie najszerszy zakres dzialan objawia sie klientowi dopiero po zalogowaniu. Klient otrzymuje wiadomosc zwrotna o stanie realizacji zadania od serwera.
